# Архітектурні рішення GamerIS
- **Frontend**: React 18 з використанням Hooks для управління станом.
- **Data Flow**: Дані отримуються через Google Apps Script (GAS), який виконує роль проксі-сервера між Google Sheets та клієнтом.
- **UI**: Tailwind CSS для забезпечення адаптивності за принципом Mobile-First.
