# 🕹️ GamerIS Pro — Інформаційна система магазину

## 📝 Опис проєкту
Вебдодаток для автоматизації геймерського магазину: каталог товарів, розрахунок вартості в кошику та перевірка залишків. Проєкт підготовлено як базу для бакалаврської роботи.

## 🚀 Стек технологій
- **React 18 + Vite** (Frontend)
- **Tailwind CSS** (Стилізація)
- **Vitest** (Автоматизоване тестування)

## 📂 Структура репозиторію
- `src/` — Вихідний код програми.
- `src/utils/` — Бізнес-логіка та UNIT-тести.
- `public/` — Статичні ресурси.
- `.gitignore` — Налаштування ігнорування системних файлів.
- `LICENSE` — Ліцензія MIT.

## 🛠️ Як запустити
1. `npm install`
2. `npm run dev`
3. Запуск тестів: `npx vitest run --coverage`
